## 🤖 Regência de Agentes de IA

Este repositório opera como **maestro de agentes de IA** em um ecossistema multi-repositório. O modelo de governança é:

| Componente | Descrição |
|---|---|
| **AGENTS.md Master** | Instruções padronizadas para agentes em todos os repositórios irmãos |
| **FRENTES_DE_TRABALHO.md** | Mapa de atuação — cada frente com escopo, triagem e regras de fronteira |
| **Memória persistente** | Fatos duráveis sobre o ambiente, preferências, convenções |
| **Skills** | Workflows reutilizáveis por tipo de tarefa (editais, fichas, catalogação) |
| **Busca vetorial** | Chromadb + all-MiniLM-L6-v2 — busca semântica 100% local em 7+ repositórios |

**Regra de ouro:** Cada repositório irmão tem seu AGENTS.md derivado deste master. Mantenha a hierarquia: Mentoria (maestro) → AGENTS.md irmãos → execução local.

## 🌍 Tecnologia Takwara — PU Vegetal + Bambu

A Tecnologia Takwara é o conjunto de soluções que combinam:

- **Poliuretano Vegetal de Mamona** (Imperveg) — impermeabilizante, selante, biocompósito, não tóxico, biodegradável
- **Bambu Tratado** (processo MPTDF) — estruturas, habitação, equipamentos, embarcações
- **Tecnologias integradas** — forno ecológico, biochar, geodésicas, saneamento ecológico, catamarã vegetal, **prancha de surf vegetal**

Esta tecnologia é o fio condutor de todos os projetos do ecossistema — da bioeconomia amazônica à mentoria, dos editais FINEP às pranchas de surf vegetais.
