## 1. Níveis de Competência do Pretenso Aluno

A anamnese deve classificar o candidato em um dos níveis abaixo para decidir se aceita ou recusa a mentoria.

### NÍVEL 1 — Iniciante (Geralmente Recusado para Mentoria)
**Perfil:** Pessoa física sem experiência com bambu, sem ferramentas, sem bancada.

**Indicadores:**
- Nunca trabalhou com bambu
- Não tem ferramentas básicas (serra, furadeira, lixadeira)
- Não tem local ventilado para aplicação de resinas
- Acha que "bambu é tudo igual"
- Não sabe diferença entre espécies (Bambusa, Guadua, Phyllostachys)

**Decisão:** ❌ Recusar. Sugerir conteúdo gratuito (e-book, vídeos demonstrativos) e reavaliar em ciclo futuro.

### NÍVEL 2 — Básico (Aceito para Mentoria Base)
**Perfil:** Pessoa física ou profissional que já teve contato com bambu, tem ferramentas mínimas e local adequado.

**Indicadores:**
- Já manuseou bambu (corte, transporte, armazenamento)
- Tem ferramentas básicas: serra, furadeira, lixadeira, trena, esquadro
- Tem local ventilado (garagem, quintal, oficina) para aplicar resinas
- Sabe identificar se o bambu está seco
- Compreende que bambu não é madeira e requer técnicas específicas
- Tem interesse específico: construir um móvel, impermeabilizar uma peça, fazer uma conexão

**Decisão:** ✅ Aceito para **Camada Base — Unboxing PU Vegetal / Espuma**
**Infraestrutura necessária:** Bancada ou mesa robusta, EPI básico (luvas, óculos), ventilação cruzada

### NÍVEL 3 — Intermediário (Aceito para Mentoria de Conexões / Movelaria)
**Perfil:** Já aplicou PU Vegetal ou tem experiência com resinas e construção.

**Indicadores:**
- Já fez a mentoria base OU já trabalha com resinas (epóxi, poliuretano)
- Tem bancada de trabalho (mesmo que improvisada)
- Sabe fazer cortes em ângulo no bambu
- Tem interesse específico: construir uma estrutura (pórtico, treliça, banqueta), fazer conexões
- Tem ferramentas: serra tico-tico ou circular, furadeira com serra-copo, grampos/sargentos

**Decisão:** ✅ Aceito para **Solda Vegetal / Conexões Estruturais**
**Infraestrutura necessária:** Bancada de gabarito (mesa plana 1,5m x 0,8m mín.), grampos tipo C, serra de precisão, EPI completo

### NÍVEL 4 — Avançado (Aceito para Domos / Inovação Takwara)
**Perfil:** Profissional da construção civil (arquiteto, engenheiro, mestre de obras) com experiência em estruturas.

**Indicadores:**
- Tem formação técnica ou experiência comprovada em construção civil
- Já trabalhou com estruturas (madeira, aço, concreto)
- Tem acesso a forno ecológico ou autoclave OU disposição para construir um
- Tem interesse específico: domo geodésico, reforço de colmos, sistema de cabos
- Tem equipe ou ajudantes para montagem
- Tem projeto definido (local, dimensões, finalidade)
- Tem infraestrutura: bancada de gabarito profissional, ferramentas elétricas, EPI completo, espaço para cura

**Decisão:** ✅ Aceito para **Domos Geodésicos** ou **Inovação Takwara (PU Expansivo + Cabo Aço)**
**Infraestrutura necessária:** Bancada de gabarito modular (mín. 3m linear), furadeira de bancada, serra circular, forno ecológico (ou acesso a tratamento térmico), EPI avançado (respirador para PU)

## 2. Infraestrutura Mínima por Tipo de Mentoria

| Mentoria | Infraestrutura Obrigatória | Infraestrutura Recomendada |
|----------|---------------------------|---------------------------|
| **Unboxing PU (Base)** | Bancada ou mesa robusta, local ventilado, EPI (luvas, óculos) | Lixadeira, furadeira, trena |
| **Solda Vegetal (Conexões)** | Bancada plana 1,5m+, grampos tipo C, serra, furadeira, EPI completo | Bancada com gabarito, serra circular, serra-copo |
| **Movelaria / Objetos** | Bancada, ferramentas de marcenaria, EPI | Torno, lixadeira de banda, gabaritos de ângulo |
| **Domos Geodésicos** | Bancada modular 3m+, furadeira de bancada, serra circular, cabos de aço, sapatilhas/premsa-cabo, forno ecológico | Gabarito de ângulos, mesa rotativa, talha para içar |
| **Inovação Takwara (Reforço)** | Bancada profissional, forno ecológico, PU Expansivo MAMONEX RD70, sistema de injeção, EPI avançado (respirador) | Compressor para injeção, formas metálicas, estufa de cura |

## 3. Conhecimentos Básicos Exigidos (Pré-requisitos por Nível)

### Para qualquer mentoria — Conhecimentos Universais
- [ ] Sabe o que é bambu e diferencia bambu de madeira
- [ ] Compreende que bambu é oco e tem nós
- [ ] Sabe que bambu precisa de tratamento para durar
- [ ] Tem noção de segurança com produtos químicos (EPI)

### Para Mentoria Base (Unboxing PU)
- [ ] Sabe selecionar bambu seco (versus bambu verde)
- [ ] Tem noção de medidas (metros, centímetros, ângulos básicos)
- [ ] Consegue seguir um passo a passo escrito
- [ ] Tem paciência para aguardar cura da resina (24-48h)

### Para Solda Vegetal / Conexões
- [ ] Já aplicou PU Vegetal (mentoria base ou experiência própria)
- [ ] Sabe fazer cortes retos e em ângulo no bambu
- [ ] Compreende o conceito de "pot life" (tempo de trabalho da resina)
- [ ] Sabe lixar e preparar superfície para adesão química
- [ ] Conhece os riscos do PU e usa EPI corretamente

### Para Domos Geodésicos
- [ ] Tem experiência comprovada com conexões de bambu
- [ ] Compreende geometria espacial básica (triângulos, ângulos, frequências v2/v3/v4)
- [ ] Sabe operar ferramentas elétricas com segurança
- [ ] Já tratou bambu termicamente (forno ecológico ou equivalente)
- [ ] Tem capacidade de ler e interpretar projeto estrutural
- [ ] Tem ajudantes para montagem (domo não se faz sozinho)

### Para Inovação Takwara (Reforço Interno + Cabo de Aço)
- [ ] Domina todas as técnicas anteriores
- [ ] Compreende sistemas de tensionamento (cabos, sapatilhas, prensa-cabo)
- [ ] Sabe dimensionar cabo de aço conforme carga
- [ ] Tem experiência com injeção de PU expansivo em moldes
- [ ] Conhece o sistema de furação angulada (45°) para passagem de cabos
- [ ] Tem infraestrutura de montagem e cura

## 6. Observações Importantes

1. **A anamnese é decisiva.** O mentor não precisa aceitar todo mundo. Recusar um aluno com nível baixo evita desgaste para ambos.

2. **Aluno recusado não é aluno perdido.** Oferecer e-book gratuito + acesso a vídeos demonstrativos + reavaliação em 3 meses.

3. **Infraestrutura é eliminatória.** Sem bancada e local ventilado, a mentoria de PU Vegetal é inviável — o risco de acidente ou mau resultado é alto.

4. **Projetos específicos (banquetas, molduras, copos, garrafas)** seguem o mesmo fluxo: vídeo demonstrativo → anamnese → mentor decide. Geralmente aceitos no Nível 3+.

5. **Presencial apenas sob demanda excepcional.** Conforme condição física do mentor (problema de coluna). Não é regra.
